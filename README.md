# Mutual TLS

## Install

```sh
go get aead.dev/mtls@latest
```

The [documentation](https://pkg.go.dev/aead.dev/mtls) contains some [examples](https://pkg.go.dev/aead.dev/mtls#pkg-examples)
for TLS clients and servers to get started.
